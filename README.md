# 🏅 120 Years of Olympic Games Analytics | Python EDA & Tableau Dashboard

## 📌 Project Overview

This project explores more than 120 years of Olympic Games history through data analysis and interactive visualizations.

Using Python for Exploratory Data Analysis (EDA) and Tableau for dashboard development, the project uncovers trends, patterns, and historical insights related to athletes, countries, sports, medals, and Olympic participation across different editions of the Games.

The goal is to transform historical Olympic data into meaningful business-style insights through data exploration and visual storytelling.

---

## 🎯 Project Objectives

- Analyze over a century of Olympic Games history.
- Explore athlete participation trends.
- Investigate medal distribution across countries.
- Identify the most successful nations and sports.
- Analyze the evolution of the Olympic Games over time.
- Build an interactive dashboard for data-driven exploration.
